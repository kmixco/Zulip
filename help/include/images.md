## Images

Here is an example of how you can add links to images that display thumbnail
previews in Zulip.

### What you type

```
[A whale of a good time](https://your.zulip.domain/user_uploads/1/46/IPvysqXEtiTG1ZdNBrwAZODi/whale-time.png)
```
