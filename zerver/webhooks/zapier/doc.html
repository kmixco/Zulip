Get notifications from every event supported by Zapier.

First, create the stream you'd like to use for Zapier notifications, and subscribe all interested parties to this stream.

{{ external_api_uri_subdomain }}/v1/external/zapier?api_key=abcdefgh&stream=zapier

Next, create a ZAP, picking the service you'd like to receive notifications from as Trigger (Step 1)

and Webhook as Action (Step 2).

As Step 2 action please choose POST

Configure Set up Webhooks by Zapier POST as follows:

  • URL is the url we created above
  • Payload Type set to JSON

Finally, configure Data. You have to add 2 fields:

  • subject is field corresponding to a subject of the message
  • content is field corresponding to a content of the message

Weather Update with Zapier!

Sign into your Zapier account, then go to the URL https://zapier.com/app/explore .
Choose the trigger app Weather by Zapier.


Next, select the option Today's Forecast.


Fill in the latitude and longitude of your location.

Next, fill in your choice of units Fahrenheit or Celsius.


After you select Webhook as the action, choose the POST option and then fill in the URL as https://[your Zulip domain]/api/v1/external/zapier?api_key=[your Zulip API key]&stream=zapier.
Select JSON as the payload type.


You have to add 2 fields:

  • subject is a field corresponding to the subject of the message, such as Here is your weather update for the day.
  • content is a field corresponding to the content of the message, such as Summary, Temperature Max, and so on.



Congratulations! You're done!
