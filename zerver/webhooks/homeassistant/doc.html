How to set up the Home Assistant integration:

In Zulip, create a homeassistant stream which will contain the received notifications. You also need a bot which you can create in the Zulip settings page.

In Home Assistant, you need to add the notify service to your configuration.yaml file. This should look somehow like this:

The api-key parameter should correspond to your bot's key. The stream parameter is not necessarily required; if not given, it will default to the homeassistant stream.

Finally, you have to trigger the service by adding an automation in your configuration.yaml file.

The data object takes at least a message property and an optional title parameter which will be the conversation topic and which defaults to homeassistant if not given.
